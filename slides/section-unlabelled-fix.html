Fix: Label not attached to input

Question:

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Consectetur, quae.

Answer:

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Consectetur, quae.

region Question: heading level 3
region Answer: heading level 3 
    
      <section aria-labelledby="question-title">
        <h3 id="question-title">Question:</h3>
        <p>Lorem...</p>
      </section>
      <section aria-labelledby="answer-title">
        <h3 id="answer-title">Answer:</h3>
        <p>Lorem...</p>
      </section>
  • The title is explicitly linked to the <section> and is now considered a landmark
  • Screen reader user can quickly jump between landmarks
  • User will get notified of a move to a new landmarks
  • Labeling other landmarks is advisable, particularly when there are multiple of the same type
  • <form> still requires role="form" to work as a landmark
  • <article> is never considered a landmark
  • Only the top level <header> and <footer> are considered landmarks
